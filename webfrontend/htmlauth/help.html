Canton Smart Soundbar Plugin

Two-way local MQTT integration for the Canton Smart Soundbar 10.

Setup

  1. Enter the soundbar IP address and click Save Configuration.
  2. The MAC address is auto-discovered via ARP on save. It is required for power_on.

MQTT State Topics (plugin → Loxone)

TopicValuesNotes
…/stateon / standbyretained
…/volume0–100retained, integer
…/muteon / offretained
…/inputinput source numberretained

MQTT Commands (Loxone → plugin)

Publish to the command topic:

PayloadAction
power_onWake soundbar via Wake-on-LAN
power_offPut soundbar in network standby
volume_set_40Set volume to 40 (0–100)
volume_upIncrease volume by step (default 5)
volume_downDecrease volume by step
mute_onMute
mute_offUnmute
mute_toggleToggle mute
input_3Switch to input source 3
